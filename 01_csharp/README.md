# 01_csharp — C# 基礎

## このフォルダで学べること

- C# の型・変数・制御構文（if / for / while / switch）
- クラス・オブジェクト・プロパティ・メソッドの設計
- 継承・抽象クラス・インターフェースによるコードの再利用
- コレクション（List・Dictionary）と LINQ を使ったデータ操作

---

## 前提知識

プログラミング未経験の方でも順番に読み進められるように構成されています。
C 言語・Java・Python などの経験がある方に向けた補足コメント（「C 経験者へ」「Java 経験者へ」など）も各ドキュメントに記載しています。
