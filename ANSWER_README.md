# answer/space-shooter ブランチについて

このブランチはスペースシューターの
動作確認済み模範解答を含む解答用ブランチです。

## このブランチに含まれるもの
- training-project/Assets/SpaceShooter/：
  Unity プロジェクトの実装済みシーン・スクリプト・Prefab
- 04_space-shooter/solution/：
  模範解答スクリプトと設計ドキュメント

## 使い方
1. このブランチを checkout する：
   git checkout answer/space-shooter
2. Unity Hub で training-project/ を開く
3. Assets/SpaceShooter/Scenes/SpaceShooterScene を開く
4. Play ボタンで動作を確認する

## 注意
このブランチは develop / main ブランチとは独立しています。
教材・ドキュメントの最新版は develop ブランチを参照してください。
